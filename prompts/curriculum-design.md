# Curriculum Design Prompt

*Use this prompt to collaboratively build your arc structure and episode topics. Paste it into a conversation in your project after setting up the project instructions.*

---

## If the AI Already Knows You

Use this version if you've been working with Claude and it already has context about your life, interests, and questions.

> I want to design a multi-arc learning curriculum using the character-based Socratic method described in my project instructions. Here's my starting direction:
>
> **What I want to explore:** [e.g., "philosophy," "creativity and meaning," "questions about purpose and legacy," "ethical questions I'm sitting with"]
>
> **General trajectory:** [e.g., "start personal and move outward," "start with foundations and build toward harder questions," "start with what I know and move toward what I'm afraid of"]
>
> **Any specific anchor points:** [e.g., "I want to end with questions about the wider world," "I want an arc about relationships," "I know I want to explore grief at some point"]
>
> Based on what you know about me and what I've shared above, propose an arc structure — 3 to 6 arcs, each with a theme and a core question. For each arc, suggest 5-10 episode topics that would build naturally from one to the next. Include which character(s) you'd recommend to lead each episode and why.
>
> This is a starting point, not a final plan. I want to react to your proposal and refine it together.

---

## If You're Starting Fresh

Use this version if you're new to the AI and it doesn't have context about your life yet. The onboarding questions build the foundation the curriculum will draw from.

> Before we design my curriculum, I want to give you context about who I am and what I'm bringing to this project. Ask me questions — about my life, my experiences, what I'm curious about, what I'm afraid of, what I've been sitting with. Ask whatever you think would help you design a curriculum that's genuinely personal to me.
>
> Don't ask everything at once. Start with 3-4 questions, and we'll go from there. I'll tell you when I feel like you have enough to work with.

*After the onboarding conversation, follow up with:*

> Now I want to design the curriculum. Here's my starting direction:
>
> **What I want to explore:** [fill in]
>
> **General trajectory:** [fill in]
>
> **Any specific anchor points:** [fill in]
>
> Based on what you've learned about me, propose an arc structure — 3 to 6 arcs, each with a theme and a core question. For each arc, suggest 5-10 episode topics. Include which character(s) you'd recommend to lead each episode and why.
>
> This is a starting point. I want to react and refine together.

---

## Tips

- Don't accept the first proposal if it doesn't feel right. Push back. "Arc 3 doesn't resonate — here's what I'm actually sitting with." The best curriculum structures emerge from negotiation
- Let the AI suggest topics you wouldn't have chosen. Some of my most productive episodes covered philosophers and themes I'd never have picked on my own
- You don't need to plan every episode before starting. Plan the first arc in detail, sketch the rest loosely, and let later arcs evolve based on what you discover
- If a proposed arc feels too abstract, ask: "What would that actually look like in a conversation about my life?" The AI should be able to make it concrete
