# Handoff Prompt

*Use this immediately after completing your crystallization. The AI generates the opening prompt for your next five-episode chat, so you never have to cold-start a new segment.*

---

## The Prompt

> Based on the crystallization we just completed, generate the opening prompt for my next chat. This prompt should:
>
> 1. **Set the context** — briefly establish where we are in the curriculum (which arc, which episode block we're entering)
> 2. **Carry forward the open questions** — the unresolved threads from the crystallization should be present as live material for the next block
> 3. **Reestablish the characters** — note any character evolution that should carry into the next episodes so the voices pick up where they left off
> 4. **Reference named patterns** — the shorthand vocabulary we've built should be available from the start
> 5. **Set up the next episodes** — based on the curriculum plan, introduce the upcoming topics and lead characters
>
> Write this as a complete, ready-to-paste prompt that I can use to start a new chat in this project. It should feel like a warm start — not a recap, but a continuation.

---

## How to Use the Handoff

1. Complete your crystallization
2. Run the handoff prompt in the same chat (while the AI still has full context)
3. Copy the generated prompt
4. Start a new chat in your project
5. Paste the handoff prompt as your first message
6. You're in — the new chat picks up where the last one left off
7. At some point, rename the chat to a consistant format (e.g. Arc 1 - Foundations (6-10))

---

## Tips

- **Review the generated prompt before pasting it.** Occasionally the AI will include something that doesn't feel right or miss something important. Edit it before using it
- **The handoff is also a good moment to adjust course.** If the planned next episodes don't feel right based on what emerged in the crystallization, this is the time to modify them
- **Save your handoff prompts.** They form a useful running log of the curriculum's progression — a trail of breadcrumbs showing how the project evolved from block to block
